## 🚀 Overview

This portfolio is built with modern web technologies focusing on **Performance**, **SEO**, and **User Experience**. It features a dynamic project showcase, a technical blog, and a sophisticated design system.

### Key Features
- **✨ Premium UI/UX**: Built with Framer Motion for fluid, buttery-smooth animations.
- **📱 Responsive Design**: Fully optimized for all screen sizes with a mobile-first approach.
- **🔍 Advanced SEO**:
  - Dynamic Metadata for every project and blog post.
  - JSON-LD Structured Data for search engine rich results.
  - Automated Sitemap and Robots.txt generation.
- **🚀 High Performance**: Leverages Next.js 15 App Router for optimized server-side rendering and static generation.
- **📝 Dynamic Blog**: Technical insights managed via a centralized data layer.

## 🛠 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)
- **SEO**: Structured Data (JSON-LD), Dynamic Metadata, Sitemap.

## 📂 Project Structure

```text
src/
├── app/            # Next.js App Router (Pages & API)
├── components/     # Reusable UI components
├── lib/            # Centralized data (projects, blog posts) and utilities
├── public/         # Static assets (images, icons)
```

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Gedions/job-orungo-portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
